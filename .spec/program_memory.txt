Header: 128 bytes.
4 bytes magic number: NNEP.
4 bytes version, starting at 0.
8 bytes code start. (128)
8 bytes data start. (Right after code. Everything before this is read only)
8 bytes file size. (Right after data. Everything after this is heap + stack)
8 bytes initial memory.

Rest is reserved.
Runnable code: From address code start to file size, padded to 8 bytes with 00.

Static memory: Memory available from the beginning.

Memory: Heap grows downwards, stack grows upwards.
